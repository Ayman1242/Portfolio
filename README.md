\# Portfolio



My personal portfolio site.



\## The Stack



\- \*\*React\*\* - component-based UI

\- \*\*Vite\*\* - dev server and build tool

\- \*\*CSS\*\* - hand-written, custom properties for theming, no framework

\- \*\*lucide-react\*\* - icons

\- \*\*Prettier\*\* - code formatting



\## Structure

src/

content.js      ← site copy (bio, jobs, projects, links)

styles.css      ← theme variables + layout styles

App.jsx         ← composes the page

components/     ← Nav, Hero, Sheet, About, Experience, Projects, Contact, Footer
